## Sistema gerenciador de estoque 

Projeto feito em java, java swing e hibernate.

Contém Cadastro, alteração e remoção de:

- Produtos
- Fornecedores
- Entradas de produtos
- Saidas

Além de estoque dinamico, consultas por id/nome/cnpj
